# Plant Assistant Frontend

Отдельный Vue 3 frontend для Laravel API. Интерфейс работает с реальными API-эндпоинтами: публичная лента доступна без авторизации, а лайки, советы, личные растения, календарь, задачи и админка требуют входа.

## Команды

```bash
npm install --cache .npm-cache
npm run dev
npm run build
```

## Настройка

```env
VITE_API_BASE_URL=/api
VITE_API_ORIGIN=http://127.0.0.1:8000
```
